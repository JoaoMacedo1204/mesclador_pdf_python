# Mesclador de PDFs com Python

## Descrição:
Script em Python que organiza e mescla arquivos em PDF em um único arquivo de forma automatizada.

## Tecnologias:
- Python
- PyPDF2

## Para executar:
1. Baixe o repositório.

2. Instale a biblioteca PyPDF2:
   pip install pypdf2.

3. Coloque os PDFs na pasta "arquivos"

4. Execute:
   python mesclador.py

## Project Insights
- Automação com Python
- Uso de bibliotecas externas
- Manipulação de arquivos
